# Calculators Tooling

This folder contains scripts for creating, embedding, deploying and validating financial calculator tools.

Files:
- `deploy_tax_calculators.py` — Deploy calculators to the site
- `embed_calculator.py` — Embed calculator iframes into pages
- `update_calculator_page.py` — Update calculator pages
- `validate_calculator.py` — Run validation tests on deployed calculators

Usage: run scripts from this directory or call them from a higher-level deploy script.
